## Packages
leaflet | Interactive maps
react-leaflet | React components for Leaflet
recharts | Data visualization (Pie chart)
jspdf | PDF generation
jspdf-autotable | Table plugin for jsPDF
framer-motion | Animations (Fade-in, transitions)
@tsparticles/react | Particle background
@tsparticles/slim | Lightweight particle engine
clsx | Utility for constructing className strings
tailwind-merge | Utility for merging Tailwind classes

## Notes
Map tiles provider: CartoDB DarkMatter (https://{s}.basemaps.cartocdn.com/dark_all/{z}/{x}/{y}{r}.png)
Images: Unsplash source for city images
Icons: Lucide React (already in base stack)
Fonts: Inter, Rajdhani (for cyberpunk headers)
